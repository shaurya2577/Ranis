# Rani's California

Indo-Western clothing brand website. Small batch, handcrafted pieces.

## Stack
- Vanilla HTML/CSS/JS
- No frameworks, keeping it simple
- Mobile-first responsive design

## Features
- Shopping cart with localStorage
- Product filters & modals
- Custom order forms
- Image admin tool for mapping product photos

## Run Locally
```bash
python -m http.server 8000
# or
npx http-server
```

Then open `http://localhost:8000`

## Structure
- `index.html` - main site
- `admin.html` - image mapper tool
- `products.json` - product data
- `styles.css` - all styling
- `scripts.js` - cart, modals, forms
